## üîß Customization Guide

### Changing Colors
Edit the CSS `:root` variables in any HTML file:

```css
:root {
    --holo-cyan: #00fff9;      /* Primary accent */
    --holo-magenta: #ff00e5;   /* Secondary accent */
    --holo-purple: #b026ff;    /* Tertiary accent */
    --vr-green: #39ff14;       /* Success/active */
    --text-bright: #ffffff;    /* Main text */
    --text-dim: #666699;       /* Secondary text */
}
```

### Adding Content
Each page follows a consistent structure:

```html
<section>
    <div class="section-header">
        <h2>Your Section Title</h2>
    </div>
    <div class="card-grid">
        <div class="card">
            <span class="card-tag">Category</span>
            <h3>Card Title</h3>
            <p>Card description...</p>
            <a href="#" class="card-link">Link Text</a>
        </div>
    </div>
</section>
```

### Updating News
Edit the `getFallbackNews()` function in `index.html`:

```javascript
function getFallbackNews() {
    return [
        {
            title: "Your Title",
            description: "Your description...",
            source: "Source Name",
            url: "https://example.com",
            publishedAt: new Date()
        }
        // Add more articles...
    ];
}
```

## üêõ Troubleshooting

### Common Issues

**Q: Fonts not loading?**  
A: Check internet connection. Google Fonts requires CDN access.

**Q: Particles not animating?**  
A: Enable JavaScript in your browser settings.

**Q: Navigation not working?**  
A: Ensure all 7 HTML files are in the same folder.

**Q: Styles look broken?**  
A: Clear browser cache and reload (Ctrl+Shift+R or Cmd+Shift+R).

**Q: News section empty?**  
A: Fallback news should display. Check JavaScript console for errors.
